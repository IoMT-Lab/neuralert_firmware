◆ dpm_timer_create()

int dpm_timer_create ( char *  task_name,
char *  timer_name,
void(*)(char *timer_name)  callback_func,
unsigned int  secs,
unsigned int  reschedule_secs 
)

In DPM sleep mode, the timer count is also activated, and when the timeout event occurs, the DA16200 wakes up and calls the registered callbacke function.

Parameters
[in]task_nameName of the process that wants to use the DPM timer. It must be the same as the name registered using the dpm_app_register() function.
[in]timer_nameA string of 7 characters or less as a unique character to distinguish timer.
[in]callback_funcFunction pointer to be called when timerout occurs. If there is no function you want to call, enter NULL.
[in]secsTimeout occurrence time.
reschedule_secsPeriodic timeout occurrence time. When set to 0, timeout occurs only once with the time set as the value of secs.
Returns
Internally set Timer ID 5 ~ 15. If Timer ID > 16, it is a failure.
