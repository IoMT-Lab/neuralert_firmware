◆ CC_RndSetGenerateVectorFunc()

CCError_t CC_RndSetGenerateVectorFunc ( CCRndContext_t *  rndContext_ptr,
CCRndGenerateVectWorkFunc_t  rndGenerateVectFunc 
)

This function sets the RND vector-generation function into the RND context.

It is called as part of Arm TrustZone CryptoCell library initialization, to set the RND vector generation function into the primary RND context.

Note
It must be called before any other API that requires the RND context as parameter.
Returns
CC_OK on success.
A non-zero value from cc_rnd_error.h on failure.
Parameters
[in,out]rndContext_ptrA pointer to the RND context buffer allocated by the user, which is used to maintain the RND state, as well as pointers to the functions used for random vector generation.
[in]rndGenerateVectFuncA pointer to the CC_RndGenerateVector random-vector-generation function.
