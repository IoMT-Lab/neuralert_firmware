◆ rnd_pseudo_rand()

static int rnd_pseudo_rand ( void *  rng_state,
unsigned char *  output,
size_t  len 
)
static

This function returns random based on a pseudo random function. This means the results should be identical on all systems. Pseudo random is based on the XTEA encryption algorithm to generate pseudorandom.

rng_state shall be a pointer to a rnd_pseudo_info structure.

static

This function just returns data from rand(). Although predictable and often similar on multiple runs, this does not result in identical random on each run. So do not use this if the results of a test depend on the random data that is generated.

rng_state shall be NULL.
