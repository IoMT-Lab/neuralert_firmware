This function encrypts or decrypts AES-OFB buffer.

Note
Due to the nature of OFB, you must use the same key schedule for both encryption and decryption, so that a context is initialized with mbedtls_aes_setkey_enc() for both MBEDTLS_AES_ENCRYPT and MBEDTLS_AES_DECRYPT.
Parameters
ctxThe AES context.
lengthThe length of the data.
nc_offNot supported. Set to 0.
nonce_counterThe 128-bit nonce and counter.
stream_blockNot supported.
inputThe input data stream.
outputThe output data stream.
Returns
0 on success.
