Convert __time64_t to tm as UTC time.

Parameters

__time64_t is an alias of a fundamental arithmetic type capable of representing times as returned by function time.
Returns

Format time as string.

Parameters
[out]sPointer to the destination array where the resulting C string is copied.
[in]maxsizeMaximum number of characters to be copied to ptr, including the terminating null-character.
[in]formatC string containing any combination of regular characters and special format specifiers.
These format specifiers are replaced by the function to the corresponding values to represent the time specified in timeptr.
Returns
If the length of the resulting C string, including the terminating null-character, doesn't exceed maxsize,
the function returns the total number of characters copied to ptr (not including the terminating null-character).
Otherwise, it returns zero, and the contents of the array pointed by ptr are indeterminate.
