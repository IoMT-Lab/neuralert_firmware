This function burns all ICV assets into the OTP of the device.

The user must perform a power-on-reset (PoR) to trigger LCS change to DM LCS.

Returns
CC_OK on success.
A non-zero value from cc_prod_error.h on failure.
Parameters
[in]ccHwRegBaseAddrThe base address of CrytoCell HW register.
[in]pCmpuDataA pointer to the ICV defines structure.
[in]workspaceBaseAddrThe base address of the workspace for internal use.
[in]workspaceSizeThe size of the provided workspace. Must be at least CMPU_WORKSPACE_MINIMUM_SIZE.
