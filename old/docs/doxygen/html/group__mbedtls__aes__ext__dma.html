This function returns the IV after an AES CMAC or a CBCMAC operation.

Returns
CC_OK on success.
A non-zero value from cc_aes_error.h on failure.
Parameters
[in]operationModeThe AES mode. Supported modes are:
  • ECB
  • CBC
  • CTR
  • CBC_MAC
  • CMAC
  • OFB
[out]ivThe AES IV buffer.
[in]iv_sizeThe size of the IV. Must be 16 Bytes.

Returns
CC_OK on success.
A non-zero value from cc_aes_error.h on failure.

  • 192bits
  • 256bits
[in]encryptDecryptFlag

  • 1: Decrypt
.

  • CBC_MAC
  • CMAC
  • OFB
[in]data_sizeData size to encrypt or decrypt.
