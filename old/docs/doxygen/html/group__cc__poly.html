CIMPORT_C CCError_t mbedtls_poly ( mbedtls_poly_key  pKey,
uint8_t *  pDataIn,
size_t  dataInSize,
mbedtls_poly_mac  macRes 
)

This function performs the POLY MAC Calculation.

Returns
CC_OK on success.
A non-zero value on failure, as defined in mbedtls_cc_poly_error.h.
Parameters
[in]pKeyA pointer to the key buffer of the user.
[in]pDataInA pointer to the buffer of the input data to the ChaCha. Must not be null.
[in]dataInSizeThe size of the input data. Must not be zero.
[in,out]macResA pointer to the MAC-result buffer.
