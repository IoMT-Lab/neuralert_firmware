Detailed Description

FFC Domain parameters structure (p,q,g,{seed,genCounter}. NIST SP 800-56A sec.5.5.1.1. Max. size of structure: 2*(MaxModSize + MaxOrderSize) + 5w*4 + 40bytes = 636 bytes
