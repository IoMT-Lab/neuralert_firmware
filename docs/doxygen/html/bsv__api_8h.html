◆ CC_BsvSocIDCompute()

CCError_t CC_BsvSocIDCompute ( unsigned long  hwBaseAddress,
CCHashResult_t  hashResult 
)

This function derives the device's unique SOC_ID as hashed (Hbk || AES_CMAC (HUK)).

Note
SOC_ID is required for the creation of debug certificates. Therefore, the OEM or ICV must provide a method for a developer to discover the SOC_ID of a target device without having to first enable debugging. One suggested implementation is to have the device ROM code compute the SOC_ID and place it in a specific location in the flash memory, where it can be accessed by the developer.
Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressCryptoCell HW registers' base address.
[out]hashResultThe derived SOC ID.
