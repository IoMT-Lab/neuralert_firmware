◆ CC_FfcCreateNewGenerator()

CIMPORT_C CCError_t CC_FfcCreateNewGenerator ( CCFfcDomain_t *  pDomain,
CCRndContext_t *  pRndContext,
uint8_t  index,
CCFfcDomainTmpBuff_t *  pTmpBuff 
)

The function creates a new FFC subgroup Generator for existed FFC Domain.

  1. Assumed, that FFC domain is properly generated or imported previously and meets to the FIPS 184-4, sec. A.1.1.2 standard.
  2. The function checks input/output pointers and buffers sizes and creates new Generator according to sec. A.2.3. and sets it into Domain structure.
Returns
CC_OK on success.
A non-zero value on failure as defined cc_dh_error.h, cc_rnd_error.h.
Parameters
[in,out]pDomainpointer to FFC Domain structure.
[in]pRndContextrandom generation function context.
[in]indexindex allowing to generate some FFC generators with the same FFC parameters prime P and Order Q, existed in the domain.
[in]pTmpBuffpointer to FFC Domain temp buffer structure.

◆ CC_FfcExportDomain()

CIMPORT_C CCError_t CC_FfcExportDomain ( CCFfcDomain_t *  pDomain,
uint8_t *  pPrime,
size_t *  pPrimeSize,
uint8_t *  pGenerator,
size_t *  pGeneratorSize,
uint8_t *  pOrder,
size_t *  pOrderSize,
uint8_t *  pSeed,
size_t *  pSeedSize,
CCFfcParamSetId_t *  pFfcParamSetId,
CCFfcHashOpMode_t *  pFfcHashMode,
uint32_t *  pGenCounter,
uint8_t *  pIndexOfGenerator 
)

This function extracts FFC domain parameters from Domain structure for external using.

  1. Assumed, that FFC domain is properly generated by CC_FfcGenerateDomain or other function according to the FIPS 184-4, A.1.1.2 standard.
  2. The function checks input/output pointers and buffers sizes, converts the DH Domain parameters to big endianness output arrays (with leading zeros if exists).
  3. Note: Sizes of parameters are given by pointers, were [in/out] values are: in - buffer size, out - actual size.
Returns
CC_OK on success.
A non-zero value on failure as defined cc_dh_error.h, cc_rnd_error.h.
Parameters
[in]pDomainpointer to FFC Domain to be exported.
[out]pPrimepointer to prime modulus of the finite field (P).
[in,out]pPrimeSizepointer to prime P size in bytes.
[out]pGeneratorpointer to generator of subgroup (G).
[in,out]pGeneratorSizepointer to generator G size in bytes.
[out]pOrderpointer to the order of the generator G.
[in,out]pOrderSizepointer to order of generator Q size in bytes.
[out]pSeedoptional, pointer to the Seed, used for Domain generation; if Seed is not required, then the pointer and size should be NULL.
[in,out]pSeedSizeoptional, size of the Seed in bytes - if the Seed not exist, in the Domain, the function sets the size = 0.
