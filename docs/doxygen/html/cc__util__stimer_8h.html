int64_t CC_UtilCmpTimeStamp ( CCUtilTimeStamp_t  timeStamp1,
CCUtilTimeStamp_t  timeStamp2 
)

This function returns the elapsed time, in nano-seconds, between two recorded time stamps. The first time stamp is assumed to be the stamp of the interval start, so if timeStamp2 is lower than timeStamp1, negative duration is returned. The translation to nano-seconds is based on the clock frequency definitions described in cc_secure_clk_defs.h.

Returns
- Duration between two time stamps in nsec.
Parameters
[in]timeStamp1Time stamp of the interval start.
[in]timeStamp2Time stamp of the interval end.
