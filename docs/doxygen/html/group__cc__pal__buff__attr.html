This function purpose is to verify the buffer's attributes according to address, size, and type (in/out). The function returns whether the buffer is secure or non-secure. In any case of invalid memory, the function shall return an error (i.e.mixed regions of secured and non-secure memory).

Returns
Zero on success.
A non-zero value in case of failure.
Parameters
[in]pDataBufferAddress of data buffer.
[in]buffSizeBuffer size in bytes.
[out]pBuffNsHNONSEC buffer attribute (0 for secure, 1 for non-secure)
