This function burns all OEM assets into the OTP of the device.

The user must perform a power-on-reset (PoR) to trigger LCS change to Secure.

Returns
CC_OK on success.
A non-zero value from cc_prod_error.h on failure.
Parameters
[in]ccHwRegBaseAddrccHwRegBaseAddr The base address of CrytoCell HW registers.
[in]pDmpuDatapDmpuData A pointer to the defines structure of the OEM.
[in]workspaceBaseAddrworkspaceBaseAddr The base address of the workspace for internal use.
[in]workspaceSizeworkspaceSize The size of provided workspace. Must be at least DMPU_WORKSPACE_MINIMUM_SIZE.
